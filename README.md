Here is a complete beginner-friendly mini project structure and code for your AI-based Automated Defective Exhibit Identification System placed in a gallery. The core idea follows real defect-detection systems that use deep learning/CNNs for visual inspection and can work with pre-trained or synthetic-data-based approaches, which is useful when real defect images are limited. �
Project title

This title is suitable for a college mini project and clearly shows the use of AI for gallery maintenance and exhibit monitoring. �

What the project does
This prototype lets users upload an exhibit image, gives a simple defect analysis result, stores a few sample reports, and shows a dashboard with counts.
